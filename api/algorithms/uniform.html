Uniform¶

Defined in algorithms/uniform.hpp

namespace sarma::uniform¶

This namespace contains required functions for the uniform (checker-board) partitioning. Uniform partitioning is one of the straightforward algorithms that is used to solve mLI problem. Its dual (sarma::uniform::bal) solves the mNC problem.
