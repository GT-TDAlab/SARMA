Ordered Probe a Load (OPaL)¶

Defined in algorithms/ordered_probe_a_load.hpp

namespace sarma::ordered_probe_a_load¶

This namespace contains required functions for the Ordered Probe a Load algorithm. This algorithm doesn’t require sparse-prefix-sum data-structure. Note that, for now, this implementation is sequential.
